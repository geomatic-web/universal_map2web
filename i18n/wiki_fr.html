Universal Map2web

Version : 1.0.0 | QGIS : 3.16+

Extension QGIS pour exporter des couches vectorielles en cartes HTML interactives avec préservation de la symbologie QGIS.

Fonctionnalités

  • Export multi-couches avec styles QGIS (couleurs, épaisseurs, opacités)
  • Conservation des étiquettes (labeling) et légende automatique
  • Fonds de plan : OSM, Google Satellite, Hybrid ou personnalisé
  • Outils : géolocalisation, mesure, recherche, plein écran, minimap, impression
  • Filtrage avancé par attribut (8 opérateurs)
  • 4 thèmes visuels : Clair, Sombre, Professionnel, Coloré
  • Regroupement des points (clustering)
  • Configuration des popups par couche

Manuel rapide

1. Lancement

Cliquez sur l'icône Universal Map2web dans la barre d'outils QGIS.

2. Export

  1. Sélectionnez les couches à exporter (onglet Couches)
  2. Configurez les popups (champs à afficher au clic)
  3. Choisissez le fond de plan et le thème
  4. Personnalisez : titre, logo, couleur d'en-tête
  5. Activez les outils souhaités
  6. Cliquez sur "OK" et choisissez le dossier de destination
  7. La carte s'ouvre automatiquement dans le navigateur

3. Navigation web

Action Commande
Zoom Molette / + / -
Déplacer Clic-glisser
Voir attributs Clic sur l'élément
Réinitialiser vue Double-clic sur la carte

4. Filtrage avancé

Dans la barre latérale de la carte : sélectionnez couche → champ → opérateur → saisissez la valeur → Appliquer.

Opérateurs disponibles : = ≠ contient commence par > < ≥ ≤

5. Gestion des couches sur la carte

  • Afficher/Masquer : cocher/décocher dans la légende
  • Clustering : activer "Regrouper les points" (couches ponctuelles)
  • Légende : affichage automatique des catégories et couleurs

Configuration requise

Composant Configuration
QGIS 3.16 ou supérieure
Navigateur Chrome 90+, Firefox 88+, Edge 90+
Mémoire 4 Go RAM minimum
Connexion Internet (fonds de plan et librairies)

Bonnes pratiques

  • Limitez à 10 couches actives maximum pour des performances optimales
  • Fichiers recommandés < 50 Mo par couche
  • Utilisez le système WGS84 (EPSG:4326) pour vos données
  • Testez l'affichage dans QGIS avant l'export
