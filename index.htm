Installation

Depuis le dépôt QGIS

  1. Ouvrez QGIS
  2. Menu Extensions → Gérer et installer les extensions
  3. Recherchez "Universal Map2web"
  4. Cliquez sur Installer

Depuis GitHub

  1. Téléchargez le ZIP depuis GitHub
  2. QGIS → Extensions → Installer à partir d'un ZIP
  3. Sélectionnez le fichier ZIP téléchargé
  4. Redémarrez QGIS si nécessaire
Nécessite QGIS 3.40 ou supérieur (compatible jusqu'à QGIS 4.x). L'interface du plugin s'adapte automatiquement à la langue de votre QGIS (Français / Anglais).

Utilisation

1. Lancer le plugin

Cliquez sur l'icône Universal Map2web dans la barre d'outils de QGIS, ou ouvrez-le depuis le menu Web.

Onglet Général

  • Choisissez la bibliothèque web utilisée pour afficher la carte (Leaflet)
  • Sélectionnez le fond de carte : OpenStreetMap, Google Satellite, ou une URL de tuile personnalisée (ex. https://{s}.tile.example.com/{z}/{x}/{y}.png)
  • Activez les outils que vous souhaitez intégrer dans la carte exportée (voir Outils & Options)

Onglet Personnalisation

  • Définissez le titre de la carte affiché dans l'onglet du navigateur et l'en-tête
  • Ajoutez un logo facultatif affiché dans l'en-tête
  • Choisissez la couleur d'en-tête
  • Sélectionnez l'un des quatre thèmes d'interface (voir Thèmes)

Onglet Couches

  1. Cochez les couches à exporter (utilisez "Tout sélectionner" / "Tout désélectionner" / "Inverser" pour accélérer cette étape)
  2. Pour chaque couche, configurez les champs attributaires qui apparaîtront dans la popup lorsqu'un objet est cliqué

Onglet Avancé

  • Simplifier les géométries — réduit le nombre de sommets pour un chargement plus rapide
  • Compresser les données JSON — fichiers GeoJSON de sortie plus petits
  • Arrondir les coordonnées — réduit la précision décimale
  • Exporter en ZIP — regroupe tout l'export dans une seule archive
  • Ouvrir automatiquement dans le navigateur — démarre un serveur local et ouvre la carte une fois l'export terminé

Onglet Wiki

Documentation de référence intégrée, disponible en français ou en anglais selon la langue de votre QGIS.

Exportation

  1. Une fois tout configuré, cliquez sur "OK"
  2. Choisissez le dossier de destination
  3. La carte s'ouvre automatiquement dans votre navigateur par défaut (si activé)

Outils & Options

Chaque outil ci-dessous peut être activé ou désactivé indépendamment dans l'onglet Général avant l'export.

Géolocalisation GPSAjoute un bouton pour centrer la carte sur la position actuelle du visiteur
Mesure (distance/surface)Ajoute des outils interactifs de mesure de distance et de surface

Gestion des couches sur la carte exportée

  • Afficher / masquer une couche : cochez ou décochez sa case dans le panneau de légende
  • Regroupement des points : activez "Regrouper les points" par couche (couches de points uniquement)
  • Légende : les couleurs et catégories sont générées automatiquement à partir de votre symbologie QGIS (les rendus Simple, Catégorisé, Gradué et Basé sur des règles sont tous pris en charge)
  • Développer / Réduire tout : pliez ou dépliez rapidement chaque groupe de couches dans la légende

Thèmes & Personnalisation

Clair (Light)

Interface claire et épurée — idéale pour les captures d'écran et les présentations de jour.

Sombre (Dark)

Interface sombre à faible luminosité, bien adaptée aux fonds de carte satellite et aux longues sessions de visualisation.

Professionnel (Professional)

Style neutre et corporatif pour les rapports officiels et les cartes web institutionnelles.

Coloré (Colorful)

Couleurs vives et accentuées, idéal pour les tableaux de bord cartographiques grand public ou promotionnels.

Astuce : la couleur d'en-tête, le titre et le logo sont indépendants du thème, vous pouvez donc mélanger l'identité de votre organisation avec l'un des quatre thèmes de base.

Filtrage avancé

Lorsqu'il est activé, un panneau de filtrage apparaît dans la barre latérale de la carte exportée, permettant aux visiteurs d'interroger les objets sans utiliser QGIS.

Comment l'utiliser

  1. Sélectionnez une couche
  2. Sélectionnez un champ
  3. Sélectionnez un opérateur
  4. Saisissez ou cliquez sur une valeur
  5. Cliquez sur "Appliquer" — les objets correspondants sont mis en évidence sur la carte

Réinitialiser : cliquez sur "Réinitialiser" à tout moment pour effacer le filtre et afficher à nouveau tous les objets.

Astuces et bonnes pratiques

Gardez-le léger

Limitez les exports à environ 10 couches actives pour de meilleures performances dans le navigateur.

Taille des fichiers

Gardez chaque couche sous environ 50 Mo ; activez "Simplifier les géométries" et "Compresser les données JSON" pour les grands jeux de données.

Système de coordonnées

Utilisez WGS84 (EPSG:4326) pour votre projet — les fonds de carte Leaflet attendent des coordonnées géographiques.

Prévisualiser d'abord

Vérifiez la symbologie et les étiquettes dans QGIS avant d'exporter — la carte web reflète le style actuel de votre projet.

Popups

Sélectionnez uniquement les champs dont les visiteurs ont réellement besoin dans la configuration des popups — moins de champs signifie une popup plus légère et plus propre.

Partager l'export

Utilisez "Exporter en ZIP" pour obtenir une archive unique que vous pouvez télécharger sur un serveur web ou envoyer à un collègue.

Dépannage

La carte ne s'ouvre pas automatiquement après l'export

Symptôme : l'export se termine sans erreur, mais aucune fenêtre de navigateur n'apparaît.
Solutions :
  • Assurez-vous que "Ouvrir automatiquement dans le navigateur" est coché dans l'onglet Avancé
  • Vérifiez que votre pare-feu ou antivirus ne bloque pas le port local utilisé pour prévisualiser la carte
  • Ouvrez index.html manuellement depuis le dossier d'export dans votre navigateur

La légende ou certaines étiquettes restent dans la mauvaise langue

Solutions :
  • Assurez-vous que la langue de l'interface de QGIS est définie comme vous le souhaitez dans Paramètres → Options → Général
  • Redémarrez QGIS après avoir changé la langue pour que le plugin recharge ses traductions

Une couche n'apparaît pas dans la carte exportée

Solutions :
  • Confirmez que la couche est cochée dans l'onglet Couches de la boîte de dialogue d'export
  • Assurez-vous que la visibilité de la couche est activée dans le panneau Couches de QGIS
  • Uniquement les couches vectorielles — les couches raster ne sont pas exportées vers Leaflet par ce plugin

Les couleurs ou catégories sur la carte web ne correspondent pas à QGIS

Solutions :
  • Vérifiez que votre type de rendu (Simple, Catégorisé, Gradué, Basé sur des règles) est entièrement appliqué dans QGIS avant l'export
  • Actualisez le panneau de symbologie de la couche dans QGIS, puis réexportez
Bon à savoir : la plupart des problèmes proviennent des autorisations du navigateur/pare-feu pour le serveur de prévisualisation local, ou de l'état de visibilité/sélection d'une couche dans QGIS au moment de l'export.
